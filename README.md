## Starting ##

```bash
# Clone this project
$ git clone https://github.com/waynje/scrapy_parser_pep

# Access
$ cd scrapy_parser_pep

# Set up venv
$ python3 -m venv venv

# Activate venv
$ source venv/bin/activate
```
